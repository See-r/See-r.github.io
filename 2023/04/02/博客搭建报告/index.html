博客搭建报告

April 2, 2023, 4:09 PM

博客主题及选取原因

  1. 使用Hexo+Github搭建个人博客，主题为Ascent。符合个人审美，简洁美观。

  2. Hexo是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。把用户的markdown文件，按照指定的主题解析成静态网页，使用较简单方便。

博客页面布局及设计思路

  1. 页面由四个模块组成：Home x About x Archives x Github

    • Home:预览文章，可以选择是否精读该文章。
    • About:个人主页，介绍基本信息
    • Archives:归档功能
    • Github:跳转博主Github主页

  2. 设计思路：通过最简单、清晰的方式，对文章进行梳理、展示。

博客的功能实现及技术选择

  1. 访客和阅读数据统计，添加脚本进行配置

  2. 在文章中添加数学公式

博客样式设计及美学考量

  1. 博客背景偏暗黄色，排版简单，阅读博客时如同阅读报纸
  2. 同样适用于手机阅读，字体清晰

    解决方案：
